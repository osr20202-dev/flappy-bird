# 🐦 Flappy Bird

A Flappy Bird clone built with **Godot 4.6** as a learning project.

## 🎮 How to Play

- **Click** or **Spacebar** to flap
- Avoid the pipes
- Get the highest score!

## 🛠️ Tech Stack

- **Engine:** Godot 4.6
- **Language:** GDScript
- **Renderer:** GL Compatibility

## 📁 Project Structure

```
flappy-bird/
├── scenes/          ← Scene files (.tscn)
├── scripts/         ← GDScript files (.gd)
├── assets/
│   ├── sprites/     ← Art assets
│   ├── audio/       ← Sound effects
│   └── fonts/       ← Fonts
└── project.godot
```
